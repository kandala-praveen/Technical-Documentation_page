CSS Documentation

CSS Introduction

  1. CSS stands for Cascading Style Sheets. It is used to style and design web pages.

  2. With CSS, you can control colors, fonts, layouts, spacing, and responsiveness of a website.

List - Types of CSS:

  1. Inline CSS – Styles written inside the HTML element’s style attribute.
  2. Internal CSS – Styles written inside the <style> tag within the <head> section.
  3. External CSS – Styles stored in a separate .css file and linked using the <link> tag.
Selectors

Selectors are patterns used to target HTML elements.

  • *→ Universal selector
  • p→ Type selector
  • .class→ Class selector
  • #id→ ID selector
  • div p→ Descendant selector
  • div > p→ Child selector
  • [attr=value]→ Attribute selector
  • :hover→ Pseudo-class
  • ::before→ Pseudo-element
Box Model

Every element in CSS follows the box model.It consists of:

  1. Content → text, images, etc.
  2. Padding → space between content and border.
  3. Border → edge of the element.
  4. Padding → space between content and border.
div {
margin: 20px;
padding: 10px;
border: 2px solid black;
}
Flexbox

Flexbox is a layout model that helps align items efficiently.

.container {
display: flex;
justify-content: center;
align-items: center;
}
  • justify-content→ horizontal alignment
  • align-items→ vertical alignment
  • flex-direction→ row / column
Grid

CSS Grid allows two-dimensional layouts.

.container {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
gap: 10px;
}
Colors & Units

You can set text color, background color, and images.

h1 {
color: blue;
background-color: yellow;
}
body {
background-image: url("bg.jpg");
}
Media Queries

  • Responsive design makes websites adapt to different screens.

@media (max-width: 600px) {
body {
background-color: lightgray;
}
}
Conclusion

  • CSS is essential for modern web development. By mastering selectors, properties, layouts (Flexbox & Grid), and responsive design, you can create beautiful and user-friendly web pages.
